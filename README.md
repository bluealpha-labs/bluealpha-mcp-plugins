# BlueAlpha Plugins for Claude

Public marketplace for BlueAlpha's Claude plugins. Built for performance marketers, growth leads, and in-house Google Ads owners.

## Available plugins

- **bluealpha-marketing-plugin** — Paid media strategy and analysis for Google Ads, with MMM-backed incrementality. [Read more →](./plugins/bluealpha-marketing-plugin/README.md)

## Install

In Claude (Cowork mode or desktop), type:

```
/plugin marketplace add bluealpha-labs/bluealpha-plugins
/plugin install bluealpha-marketing-plugin
```

That's it. The first time you use a skill, Claude will prompt you to connect your BlueAlpha account.

## Don't want to use the marketplace?

You can also install the plugin from a single `.plugin` file. See the [latest release](https://github.com/bluealpha-labs/bluealpha-plugins/releases/latest) and drag the file into Claude.
