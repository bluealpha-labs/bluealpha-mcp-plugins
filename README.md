# BlueAlpha Plugins for Claude Performance Marketers

Public marketplace for BlueAlpha's Claude plugins. Built for performance marketers, growth leads, and in-house Google Ads, TikTok, Meta, X and LinkedIn owners.

## Available plugins

- **bluealpha-marketing-plugin** — Paid media strategy and analysis for Google Ads, TikTok Ads, and LinkedIn Ads, with MMM-backed incrementality. [Read more →](./plugins/bluealpha-marketing-plugin/README.md)

## Install

In Claude Code, type:

```
/plugin marketplace add https://github.com/bluealpha-labs/bluealpha-plugins.git
/plugin install bluealpha-marketing-plugin
```

That's it. The first time you use a skill, Claude will prompt you to connect your BlueAlpha account.

## Don't want to use the marketplace?

You can also install the plugin from a single `.plugin` file. See the [latest release](https://github.com/bluealpha-labs/bluealpha-plugins/releases/latest) and drag the file into Claude.
